# 2026-001_projet-web

Projet web demarre en local, puis publication sur GitHub.

## Arborescence
- 01_docs: specs, decisions, notes
- 02_source/frontend: app front
- 02_source/backend: API/services
- 03_tests: tests auto
- 04_assets: images, fonts, ressources
- 05_exploitation: deploiement, env, scripts ops
- 06_livraison: versions livrees

## Demarrage local (frontend statique)
1. Ouvrir `02_source/frontend/index.html` dans le navigateur

## Passage sur GitHub
1. Creer un repo vide sur GitHub
2. Ajouter le remote:
   git remote add origin <URL_GITHUB>
3. Pousser la branche principale:
   git branch -M main
   git push -u origin main

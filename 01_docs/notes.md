# Notes Projet

## Vision
Assistant de craft POE2 pour debutants et avances, avec conseils adaptes a chaque situation.

## V1 livree
- Formulaire de craft detaille
- Type de degats (physique, feu, froid, foudre, chaos, minions, mixte)
- Budget exact en Divines
- Recommandations de mods prioritaires par slot et objectif
- Checklist avant craft
- Plan d'action pas a pas
- Scores: puissance, risque, pression cout
- Mode debutant / avance
- Historique local
- Favoris
- Guide rapide (erreurs frequentes + routine)

## Choix techniques
- Frontend local: HTML/CSS/JS
- Stockage: localStorage
- Cout infra: 0 EUR
- Cible hebergement: GitHub Pages

## Suite conseillee (V1.1)
- Persistance cloud (quand besoin multi-device)
- Raffinement des regles par archetype de build
- Export/import JSON des plans de craft
